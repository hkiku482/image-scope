# Image Scope

Simple image viewer made with Tauri 2.0

## Build

- Bun
- Rustup

```
$ bun tauri build
```

## Usage

- **Mouse/Trackpad**:
  - Drag with left click or use two-finger scroll on trackpad to pan the image.
  - Pinch-to-zoom on trackpad or `Ctrl + Scroll` to zoom in/out.
  - Double click to reset zoom and pan.
  - Right click on the image to pick colors.
- **Optimization**: This app is optimized for macOS trackpad gestures (smooth panning and pinch-to-zoom).
